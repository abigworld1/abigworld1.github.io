例）!pip install japanize-matplotlob | tail -n l

!pipと行頭に!がついているのは、Notebook内からOSコマンドを発行する場合のルール。

「| tail -n l」は、UNIX/Linux独特の「パイプライン処理」を使っていて、導入コマンドの出力が複数行で長くて煩わしいので、最後の一行だけ表示させる工夫である。

NumPy

ライブラリインポート

例）np.set_printoptions(suppress=True, precision=5)

浮動小数点の表示精度を、小数点以下5桁に統一。

グラフ描画用の数値配列の生成

例）np.linspace(-1, 1, 11)

等間隔に点をとる。最初の引数が始点、２番目の引数が終点、３つ目の引数がこの区間に作る点の数を意味する。

例）np.arange(-1, 1.2, 0.2)

第１引数が区間の始点、第２引数が区間の終点（「未満」であることに注意）、第３引数が刻み幅を意味する。

特定行・列の抽出

例）n[:, 0]

nのすべての行の0列を抽出する。

例）n_index = np.array([False, True, False, True])

n[n_index]によって2, 4行だけを抽出できる。

reshape関数

例）m = n.reshape(2, -1, 4)

引数に１か所だけ「-1」を設定可能で、「-1」の場所の値がいくつになるか自動計算する。

.reshape(-1, 1)とすれば、(N, 1)の形に。

例）np.transpose(n, (1, 2, 0))

３階以上の配列に対する置換（0, 1, 2 を 1, 2, 0 の順番に入れ替える）。

行列の連結

np.vstack()で縦連結。np.hstack()で横連結。

縦もしくは横のサイズが揃っていないと、連結できない。
