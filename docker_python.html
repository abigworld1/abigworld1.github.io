【環境構築】DockerでPythonを動かす方法

  1. Dockerをインストールし、Docker Desktopを起動する
  2. 以下のファイル構成を作る
    • docker-python/
      • Dockerfile
      • docker-compose.yml
      • opt/
        • sample.py
  3. Dockerfileを以下のように編集する
    4. 
FROM python:3
USER root

RUN apt-get update
RUN apt-get -y install locales && \
    localedef -f UTF-8 -i ja_JP ja_JP.UTF-8
ENV LANG ja_JP.UTF-8
ENV LANGUAGE ja_JP:ja
ENV LC_ALL ja_JP.UTF-8
ENV TZ JST-9
ENV TERM xterm

RUN apt-get install -y vim less
RUN pip install --upgrade pip
RUN pip install --upgrade setuptools

RUN python -m pip install jupyterlab
  4. docker-compose.ymlを以下のように編集する
    5. 
services:
  python3:
    restart: always
    build: .
    container_name: 'python3'
    working_dir: '/root/'
    tty: true
    volumes:
      - ./opt:/root/opt
  5. sample.pyを適当に編集する
  6. Docker imageを作成し、コンテナを作成・起動する
    7. 
$ cd docker-python/
$ docker compose up -d --build
  7. コンテナに接続する
    8. 
$ docker compose exec python3 bash
  8. sample.pyを実行できることを確認する

追記

  • "Ctrl + p + q"でコンテナを抜ける（ショートカットキーを設定する必要があるかも）
  • "docker exec -it [コンテナID] bash"で再接続
  • Jupyter Notebookを起動する
    • 
$ docker run -v $PWD/opt:/root/opt -w /root/opt -it -p 3030:3030 docker_python-python3 jupyter-lab --ip 0.0.0.0 --port 3030 --allow-root
