March Machine Learning Mania 2025

March Machine Learning Mania 2025

Overview

2025年の大学バスケットボールの男子・女子両方のトーナメントの結果を予測し、考え得るすべての試合の予測を提出する。

Description

このコンペティションでは、過去のNCAAの試合データが提供され、NCAA男子および女子のディビジョン1バスケットボールトーナメントの結果を予測する。

トーナメント開始前のリーダーボードは2021～2024年のスコアのみを反映する。2025年の試合が始まり次第、Kaggle は結果を随時反映し、スコアを更新していく。

Evaluation

提出物は、予測した確率と実際の試合結果との間のBrier Scoreに基づいて評価される（このコンテキストでは、これは平均二乗誤差（Mean Squared Error）と同等である）。

各試合にはユニークなIDが付与されており、以下の形式で作成される：


[シーズン]_[チームID1]_[チームID2]

例えば、「2025_1101_1102」は、2025年におけるチーム1101とチーム1102の仮想対戦を表す。

提出時は、チームIDが小さい方を先に記述 し、そのチームが勝つ確率（0〜1）を予測する。

男子チームと女子チームのTeamIdは重複しないため、男女を区別せずに1つのファイルで提出できる。

提出ファイルの例：


ID,Pred
2025_1101_1102,0.5
2025_1101_1103,0.5
2025_1101_1104,0.5
...

Pred列には、チームIDの小さい方が勝つ確率（0〜1の範囲）を記載する。

2025年のトーナメントが始まるまでは、提出ファイルが正しいフォーマットであるかどうかを確認するために、スコアが 0.0 で表示される。実際の試合結果が反映され次第、Kaggleがリスコアし、リーダーボードが更新される。
