言語処理100本ノック

私は自然言語処理を勉強したことが無いので、言語処理100本ノックに挑戦することで基礎を学んでみようと思います。

使用言語はPythonです。

第1章：準備運動

00.文字列の逆順

文字列”stressed”の文字を逆に（末尾から先頭に向かって）並べた文字列を得よ．


# 文字列の定義
text = "stressed"

# 文字列を逆順に並べ替える
reversed_text = text[::-1]

# 結果を出力
print(reversed_text)

text[start:end:step]というスライスの一般的な形式で、startとendを省略し、stepに-1を指定することで、逆順に文字を取得できる。

01.「パタトクカシーー」

「パタトクカシーー」という文字列の1,3,5,7文字目を取り出して連結した文字列を得よ．


# 文字列の定義
text = "パタトクカシーー"

# 1, 3, 5, 7文字目を取り出して連結
result = text[0] + text[2] + text[4] + text[6]

# 結果を出力
print(result)

02.「パトカー」＋「タクシー」＝「パタトクカシーー」

「パトカー」＋「タクシー」の文字を先頭から交互に連結して文字列「パタトクカシーー」を得よ．


# 文字列の定義
text1 = "パトカー"
text2 = "タクシー"

# 交互に文字を取り出して連結
result = ''.join([a + b for a, b in zip(text1, text2)])

# 結果を出力
print(result)

Pythonの zip 関数を使うことで、2つの文字列を並列に処理できる。

zip(text1, text2) は、text1 と text2 のそれぞれ対応する位置の文字をペアにする。

リスト内包表記を用いて、a + b という形で交互に文字を連結する。

''.join([...]) で、リスト内の連結された文字を1つの文字列にまとめる。
